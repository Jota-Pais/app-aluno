# App Aluno - Frontend

Este é um repositório de um projetinho da aula de frontend do curso de engenharia de software, disciplina de frontend.

## Descrição

O projeto consiste em uma aplicação web simples para autenticação e cadastro de usuários, desenvolvida com HTML puro e CSS. A aplicação simula um portal acadêmico onde alunos podem fazer login, recuperar senha e criar novas contas.

## Estrutura do Projeto

- **index.html** - Página principal de login com formulário de autenticação
- **cadastro-usuario.html** - Primeira etapa do cadastro (coleta de CPF)
- **cadastro-usuario-dados.html** - Segunda etapa do cadastro (dados pessoais completos)
- **recuperar-senha.html** - Página para recuperação de senha via e-mail
- **style.css** - Arquivo de estilos

## Funcionalidades

- Login de usuários com e-mail e senha
- Cadastro em duas etapas (CPF → dados pessoais)
- Recuperação de senha por e-mail
- Interface responsiva e amigável
- Design educacional com citações inspiradoras

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Como Usar

1. Clone este repositório
2. Abra o arquivo `index.html` em seu navegador
3. Navegue pelas diferentes páginas para testar as funcionalidades
